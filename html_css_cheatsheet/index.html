CSS Cheatsheet

CSS Properties to Change the Appearance of Text

CSS Property

Function

Possible Values

Result
font-family Specifies the typeface. "Courier New", Courier, monospace Hello, World!
font-size Specifies the size of the font. 16px 200% 1.5em Hello, World!
font-weight Used to create bold text. bold normal Hello, World!
font-style Used to create italic text. italic oblique Hello, World!
text-transform Used to change the case of text. uppercase capitalize Hello, World!
text-decoration Used to add lines above, underneath, or through text. underline line-through Hello, World!
letter-spacing Used to change the space between letters. 0.2em Hello, World!
word-spacing Used to change the space between words. 1.5em Hello, World!
text-align Used to change the alignment of text. left center Hello, World!
text-indent Used to add indentation to text. 50px Hello, World!
line-height Used to change the vertical space between lines of text. 1.5em Hello,
World!
text-shadow Used to add shadow effects to text.
  • The first value refers to the width of the shadow;
  • the second one to its height;
  • the third one to its blur;
  • and the fourth to its color.
